Setup Guide

Install the extension & connect Gmail

One-time setup takes about 5 minutes. You need Google Chrome and a Gmail account.

Download Recon Extension (.zip)
  1. Download the extension

    Click the button above. Unzip the file — you should get a folder called recon-extension.

  2. Load it in Chrome

    Open chrome://extensions in Chrome.

    • Turn on Developer mode (top-right toggle)
    • Click Load unpacked
    • Select the recon-extension folder you unzipped
    • Pin the Recon icon in your Chrome toolbar

    Developer mode is the toggle in the top-right. Then use Load unpacked on the left.

  3. Generate your API key

    Each person gets their own key — do not share yours.

    Go to Extension Key →
  4. Configure the extension popup

    Click the Recon icon in Chrome. The API Server URL is already filled in — you only need to paste your API key from step 3.

    Click Save. You should see “Logged in as {{ user.email }}”.

  5. Send a test email
    • Open Gmail and compose a new email
    • Send to an external address (not your own inbox)
    • Have the recipient open it
    • Check the dashboard (hit Refresh) or Gmail Sent checkmarks
